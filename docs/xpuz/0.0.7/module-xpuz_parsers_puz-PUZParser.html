Class: PUZParser

xpuz/parsers/puz~ PUZParser

new PUZParser()

Parser class for PUZ-formatted puzzles.

Methods

generate(puzzle [, options])

Given a Puzzle object, returns a Buffer containing the puzzle in .puz format.

Parameters:
Name Type Argument Description
puzzle module:xpuz/puzzle~Puzzle

the puzzle to convert to .puz content.
options object <optional>

an object containing additional options for the conversion

Properties
Name Type Argument Description
scrambled boolean <optional>

if true, the puzzle's solution will be scrambled
solutionKey Number <optional>

the solution key with which to scramble the solution. If options.scrambled is true, this is required.
Throws:

if options.scrambled is true but options.solutionKey is not a 4-digit integer (between 1000 and 9999, inclusive).

Returns:

a Buffer containing the .puz content.

Type
external:Buffer

parse(path [, options])

Parses a file in .puz format into a Puzzle object.

Parameters:
Name Type Argument Description
path string | external:Buffer | ArrayBuffer

the .puz file to parse, either as a file path (strong) or a Buffer or ArrayBuffer containing the puzzle content.
options object <optional>

an object of options to affect the parsing

Properties
Name Type Argument Description
solutionKey Number <optional>

an integer between 1000 and 9999, inclusive, to use to unlock the puzzle's solution if the solution is locked. If the solution is not locked, this is ignored.
Throws:

if the puzzle is locked and an invalid (or no) options.solutionKey was provided

Returns:

a promise that resolves with the Puzzle object

Type
external:Q/Promise
